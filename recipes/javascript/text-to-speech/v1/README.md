# JavaScript / Text-to-Speech / v1

## Recipes

| Recipe | Description |
|--------|-------------|
| [generate-audio](generate-audio/) | Convert text to audio and save as a file using the Deepgram TTS REST API. |
| [stream-audio](stream-audio/) | Stream TTS audio as it generates via the REST API. |
| [websocket-streaming](websocket-streaming/) | Low-latency text-to-speech via WebSocket for real-time use cases. |
| [select-model](select-model/) | Choose from available aura-2 voice models for speech synthesis. |
| [select-encoding](select-encoding/) | Choose the output audio encoding format for TTS audio. |

## Notes

All recipes in this directory require `DEEPGRAM_API_KEY` to be set in the environment.
