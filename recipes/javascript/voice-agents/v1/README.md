# JavaScript / Voice Agents / v1

## Recipes

| Recipe | Description |
|--------|-------------|
| [connect](connect/) | Establish a WebSocket voice agent session with default settings. |
| [custom-llm](custom-llm/) | Use a custom LLM provider for the voice agent's "think" step. |
| [custom-tts](custom-tts/) | Choose a specific aura-2 voice model for the agent's speech output. |
| [function-calling](function-calling/) | Inject tool calls for the LLM to use during a voice agent conversation. |

## Notes

All recipes in this directory require `DEEPGRAM_API_KEY` to be set in the environment.
