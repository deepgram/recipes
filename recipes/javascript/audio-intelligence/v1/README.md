# JavaScript / Audio Intelligence / v1

## Recipes

| Recipe | Description |
|--------|-------------|
| [summarize](summarize/) | Generate a concise text summary of spoken content. |
| [sentiment](sentiment/) | Segment-level positive/negative/neutral sentiment scoring. |
| [topics](topics/) | Identify key topics discussed in audio content. |
| [intents](intents/) | Detect caller/speaker intents from audio context. |
| [entities](entities/) | Identify named entities: people, organisations, locations in audio. |

## Notes

All recipes in this directory require `DEEPGRAM_API_KEY` to be set in the environment.
