## Code Samples

This repository is the canonical content source for Deepgram code samples and example documentation. It prioritizes clear, copy/paste-ready guidance; any automation present exists only to keep content current.

### What’s included

- **docs/**: Human-readable example guides and snippets, organized by product and language.
- **.github/**: Minimal automation to refresh content and open pull requests when updates are detected.

### Contributing

- Focus on content quality and clarity in `docs/`.
- Avoid repository-specific run instructions inside example docs.
- Never commit secrets or environment files.
